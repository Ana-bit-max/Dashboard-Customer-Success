# 📊 Dashboard de Customer Success - SaaS Metrics

Este projeto é um dashboard interativo para visualização de métricas essenciais em empresas SaaS (Software as a Service), como Churn Rate, NPS e Taxa de Retenção.

## 🚀 Tecnologias Utilizadas

*   **HTML5**: Estruturação semântica.
*   **Tailwind CSS**: Estilização moderna e responsiva.
*   **JavaScript (ES6+)**: Lógica e manipulação do DOM.
*   **Chart.js**: Biblioteca para renderização de gráficos dinâmicos.

## 📈 Métricas Monitoradas

1.  **Churn Rate**: Percentual de clientes que cancelaram o serviço no período.
2.  **Taxa de Retenção**: Percentual de clientes que permaneceram ativos.
3.  **NPS (Net Promoter Score)**: Indicador de lealdade e satisfação do cliente.
